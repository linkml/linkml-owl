# Mapping Vocabulary

Use the following keywords to annotate your schema elements:

 * Slots
    - Axiom type designators
        - SubClassOf
        - EquivalentClasses
        - AnnotationAssertion
        - ...
    - Expression modifier designators
        - ObjectSomeValuesFrom
        - ObjectAllValuesFrom
        - ...
    - Collection type designators
        - ObjectIntersectionOf
        - ObjectUnionOf
        - ...
 * Classes
    - Class
    - Individual
    - ...
